# TFG: Vision-Language Model para Informes Clínicos

Proyecto de Fin de Grado en Ingeniería Informática (ETSIIT, UGR).
Desarrollado por Rafael Carrillo Arroyo.

## Estructura del Proyecto
- `notebooks/`: Cuadernos de entrenamiento y experimentación (Módulo 1 y 2).
- `data/`: Carpeta local para los datasets (excluida de Git).
- `checkpoints/`: Pesos de los modelos entrenados (excluidos de Git).
- `outputs/`: Resultados de evaluación, tablas y mapas de calor (excluidos de Git).
